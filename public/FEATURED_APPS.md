# Featured Nostr Apps

This file contains the featured apps displayed on the home page and featured sections. You can easily edit this file to add, remove, or update featured apps without touching any code.

## How to Edit

Each featured app follows this format:
```
## App Name
- **URL**: https://example.com
- **Description**: Brief description of what this app does
- **Category**: Social | Client | Tools | Media | Gaming | Marketplace
- **Platform**: Web | iOS | Android | Desktop | All
- **Icon**: https://example.com/favicon.ico
```

**Icon Options:**
- **Remote URLs**: Use `https://website.com/favicon.ico` for actual website icons
- **Lucide Icons**: Use icon names like `smartphone`, `globe`, `zap`, `users`, `play`, `shopping-cart`, `camera`, `message-circle`, `heart`, `star`, `trending-up`, `music`
Available categories: `Social`, `Client`, `Tools`, `Media`, `Gaming`, `Marketplace`, `Developer`
Available platforms: `Web`, `iOS`, `Android`, `Desktop`, `All`
