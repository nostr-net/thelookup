# Nostr Resources

This file contains the resources displayed on the `/resources` page. You can easily edit this file to add, remove, or update resources without touching any code.

## How to Edit

Each resource follows this format:
```
## Resource Name
- **URL**: https://example.com
- **Description**: Brief description of what this resource does
- **Category**: Official | Relay | Tools | Client | Gateway
- **Icon**: globe | users | zap | book-open | shield | hash | search
```

Available icons: `globe`, `users`, `zap`, `book-open`, `shield`, `hash`, `search`
Available categories: `Official`, `Relay`, `Tools`, `Client`, `Gateway`
